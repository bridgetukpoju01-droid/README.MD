
# 💰 SmartSpend  

SmartSpend is an **AI-powered personal budgeting app** that helps users track expenses, analyze spending patterns, and make smarter financial decisions.  

---

## 🚀 Features  
- 📊 **Expense Tracking** – Log and categorize daily expenses.  
- 🤖 **AI Insights** – Get personalized budgeting tips powered by AI.  
- 💡 **Smart Recommendations** – Detect overspending trends and suggest savings strategies.  
- 📱 **Dashboard Overview** – Visualize income, expenses, and savings.  
- 🔔 **Reminders & Alerts** – Notifications for bills, budgets, and overspending.  

---

```bash

# Navigate into the folder
cd smartspend
